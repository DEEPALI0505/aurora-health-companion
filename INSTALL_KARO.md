# 🌿 Aurora App — Install Guide

## Yeh app kya hai?
Aurora ek AI-powered health companion app hai jisme:
- 💧 Water tracking (bottle animation)
- 🌙 Sleep tracker
- ⚡ Habit tracker  
- 🥗 Nutrition tracker
- 🧠 AI predictions (Aurora Bot)
- 🔔 Smart notifications
- ✨ Liquid Glass UI

---

## 📱 Install karne ke 2 tarike:

---

### Tarika 1 — Expo Go (Sabse Aasaan - FREE)

1. **Phone pe install karo:** Play Store mein "Expo Go" search karo → install karo
2. **PC pe yeh folder kholo** → Terminal/Command Prompt kholo
3. **Run karo:**
```
npm install
npx expo start
```
4. **QR scan karo** Expo Go se → Aurora phone pe!

> ⚠️ Phone aur PC ek WiFi pe chahiye

---

### Tarika 2 — APK Build (Permanent Install)

**Windows pe:**
1. `BUILD_APK_WINDOWS.bat` file pe double-click karo
2. Expo account banao (free) jab bole
3. 15 min baad APK link milega
4. APK download karo → phone pe install karo

**Mac/Linux pe:**
```bash
bash BUILD_APK_MAC_LINUX.sh
```

---

## 🔧 Requirements
- Node.js (nodejs.org se free download)
- Internet connection
- Android phone (iOS ke liye Apple account chahiye)
