# 🌿 Aurora — Share & Install Guide

## Kisi ko bhi bhejna hai? Ye karo:

### Option 1 — Expo Internal Distribution Link
Build ke baad expo.dev pe jaao:
- Projects → Aurora → Builds
- "Install" button → QR code ya direct link milega
- Woh link share karo — phone pe browser se open karein → Install!

### Option 2 — APK File Direct Share
`eas build --platform android --profile preview` ke baad:
- APK download karo expo.dev se
- WhatsApp / Google Drive / Email se bhejo
- Recipient: "Unknown sources" allow karo → Install karo

### Option 3 — Google Play Store (Public)
Sab ko available hoga - needs ₹2000 one-time fee

---

## Recipient ke liye steps (jo install karega):
1. Link/APK receive karo
2. Android Settings → Security → "Install unknown apps" → ON
3. APK tap karo → Install
4. Aurora open karo → Sign Up → Apna naam daalo
5. Enjoy! 🌿
